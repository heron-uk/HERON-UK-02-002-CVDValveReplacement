# Study diagnostics

1. Open in Rstudio the diagnostics_code.Rproj project
2. Rebuild the R-environment from the provided lock-file in renv.lock with `renv::restore()`.
3. Fill the infromation required in the script `CodeToRun.R` and set the code running the last line of the script `source(here("run_study.R"))`.
4. When finished, you should expect a `.csv` file in the results folder. You can view locally (see `diagnostics_shiny` directory) and then share this csv file with your results.
